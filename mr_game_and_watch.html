Mr. Game & Watch


Occupation: Diver, Chef, Lion Tamer, Construction worker, Firefighter

Quote: Beep boop beep

Appearances: Game and Watch game systems


Bio: Mr. Game & Watch is a generic amalgam of characters from the Game & Watch games. This line of handheld electronic games produced by Nintendo from 1980 to 1991. Created by game designer Gunpei Yokoi, each Game & Watch features a single game to be played on an LCD screen in addition to a clock, an alarm, or both
